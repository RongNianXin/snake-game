# 变更日志 (Changelog)

所有本项目的版本变更都将记录在此文件中。

## [2.1] - 2026-06-08

### 新增 (Added)
* **独立词库系统**：
  * 新增 `poetry-data.js` 外部词库文件，与 HTML 代码完全解耦。
  * 词库挂载在 `window.CUSTOM_POETRY_BANK`，支持玩家二创扩展，无需修改源码。
  * ID 百位分段编号（1~99 classic / 101~199 martial / 201~299 develop），互不冲突。
* **题材选择器**：
  * 封面新增横向卡片式题材选择组件（墨香诗词阁 / 刀光剑影录 / 开发者自嘲 / 全部随机）。
  * Tailwind 青绿高亮选中态，带平滑过渡动效。
  * 选中题材后，游戏从对应 `category` 号段随机抽诗。
* **词库史诗级扩容**：
  * 总计 **60 首**（classic 20 / martial 20 / develop 20），全部逐条校验 POETRY_RULES.md 平仄押韵。
  * classic：杜甫、王维、李白、王勃等唐代律诗名联，拒绝小儿科。
  * martial：快意恩仇、塞外烽烟、铁马金戈。
  * develop：软件 Bug / 嵌入式烧板 / EDA 布线 / 电容炸飞 / 炼丹调参 / 实验室通宵，理工科大共鸣。
* **POETRY_RULES.md 声韵规范**：
  * 上句尾仄声（三四声）、下句尾平声（一二声）、禁止同音撞韵。
  * 面向 AI 的少样本提示模板 + 玩家二创自检清单。
* **Space-Aware A* 寻路大脑**：
  * Phase 1：A* 觅食 + 虚拟未来模拟（simulateEat），验完再走。
  * Phase 2：食物不安全时自动追尾兜圈子。
  * Phase 3：洪水填充计算邻域空格最大值，引导蛇向最空旷处逃生。
  * 蛇尾保护红线：短蛇或头尾 ≤ 2 格时封锁尾巴周边防止自残。
  * 测试模式 A* 迭代上限提升至 6000 次。
* **多步超频压测**：测试模式下单帧执行 8 步 AI 寻路 + 移动，配合 8 倍速实现 ~64 倍暴走。
* **排行榜单条删除**：每行记录右侧新增 × 按钮，支持精准删除单条测试记录。
* **测试数据隔离**：测试模式下成绩不写入 localStorage 排行榜。
* **排行榜一键清空**：标题右侧垃圾桶按钮，清空全部历史记录。
* **死亡瞬间定格（Post-Mortem）**：游戏结束时 Canvas 画面原样保留蛇身姿态，不叠加遮罩弹窗，画布上方显示红色"游戏结束"+ 分数。
* **游戏结束按钮重构**：按钮移至 Canvas 下方居中，透明背景青绿边框卡片样式。
* **键盘游戏结束导航重写**：`currentFocusBtn` 状态机 + 直接 DOM style 操作白色焦点边框，← → 切换，Enter/Space 确认。

### 修复 (Fixed)
* **蛇尾碰撞致命 Bug**：`step()` 中 `onSnake(h)` 在 `snake.pop()` 之前执行，AI 算法允许进入尾巴格但游戏规则判定为自杀。改为：进入尾巴格且本步不吃食物时合法（尾巴会移走）。
* **Phase 3 洪水填充全废**：`floodFill` 起点被错误加入障碍集，返回值恒为 0，所有候选方向得分一致退化为随机选。改为 `obsSet.delete(nh)` 排除起点。
* **洪水填充上限过小**：`count < 300` 导致 625 格棋盘上大空间全饱和为同一分数。去掉上限。
* **Phase 2 最短路追尾缠死**：A* 最短路径到尾巴导致头尾机械贴近、身体在中央碎片化乱绕。改为对所有安全方向洪水填充评分 + 距离尾巴奖励分，选空间最大且远离尾巴的方向。
* **键盘死锁**：测试模式下死亡时 `finishGame()` 强制 `IS_TEST_MODE = false`，`hKey()` 改为 `if (IS_TEST_MODE && !gameOverFlag) return`。
* **名称输入框光标消失**：`click` 事件冒泡拦截。
* **游戏结束键盘穿透**：Space 键不再在游戏结束时提前 `return`。
* **排行榜 × 按钮不可见**：对比度提升至纯红 `#ef4444`，hover 放大 1.2 倍。

### 优化 (Optimized)
* `initNextPoetry()` 支持按 `currentCategory` 过滤 + 排除已通关诗 + 空池兜底。
* Phase 2/3 全面重构：最短路追尾 → Space-Aware 游走评分（洪水填充 + 远离尾巴 + 贴墙贴身）。
* 声韵规则全面修正 martial 和 develop 分类诗句。


## [2.0] - 2026-06-07

### 新增 (Added)
* **诗词汉字核心玩法**：
  * **汉字食物**：食物从方块升级为单个汉字，Canvas 金色楷体渲染，带呼吸光晕。
  * **诗词词库**：内置 3 首五言诗（静夜思、悯农、春晓），每局随机抽取。
  * **去重生成**：食物智能去重，同一首诗不会重复出现相同汉字。
* **汉字收集面板**：
  * 右侧新增「当前收集进度」面板，5 个田字格（水墨十字虚线边框）。
  * 吃到的汉字依次填入格子，集满后自动按诗句顺序归位重排。
* **水墨风结算弹窗**：
  * 宣纸色背景 + 红色「詩」字印章 + 大字诗句展示。
  * 通关后蛇身定格、弹窗浮现，支持 `Enter` 键或点击进入下一关。
* **无缝换关机制**：
  * 点击「下一关」后仅清空田字格，保留蛇身长度、当前得分与速度。
  * 自动切换至不同的新诗，长蛇原地继续追逐新汉字。
* **自动化跑测模式**：
  * **URL 激活**：访问 `?test=true` 自动开启 AI 跑测（免改源码）。
  * **F8 彩蛋**：游戏中按 `F8` 实时切换测试模式，屏幕 Toast 提示。
  * **AI 寻路**：极简贪心算法自动朝食物移动。
  * **8 倍速驱动**：测试模式下游戏速度提升 8 倍。
  * **自动通关**：集齐 5 字后 500ms 自动触发下一关，实现无人值守无限跑测。
* **结算弹窗键盘支持**：结算弹窗显示时按 `Enter` 键可直接进入下一关。

### 优化 (Optimized)
* **得分与蛇身继承**：换关时得分累加、蛇身不断，难度速度持续递增，游戏寿命大幅延长。

### 修复 (Fixed)
* **基础逻辑保留**：v1.2 全部功能（难度系统、玩家系统、键盘导航、粒子特效）在 v2.0 中完整继承。


## [1.2] - 2026-06-07

### 新增 (Added)
* **布局重构**：
  * **面板整合**：得分和最高分移至右侧面板，与得分记录整合在同一列显示。
  * **视野扩大**：画布尺寸从 500px 扩大到 650px（25格×26px），蛇的活动空间增加 69%。
  * **信息丰富**：右侧新增当前玩家名称显示。
* **玩家名称系统**：
  * **名称输入**：开始画面新增名称输入框，支持中英文输入。
  * **随机命名**：提供 25 种趣味随机名称（骰子按钮）。
  * **本地缓存**：已参与过的玩家自动保存，下次可直接点击选择。
  * **记录管理**：支持删除已保存的玩家记录。
  * **追溯优化**：历史记录每条显示对应玩家名称。
* **游戏结束画面优化**：
  * **流程优化**：新增”退出”按钮，可返回开始画面切换玩家。
* **菜单键盘导航**：
  * **暂停菜单**：使用 `↑` `↓` 键切换选项，`Enter` 键确认选中项。
  * **结束菜单**：使用 `←` `→` 键切换选项，`Enter` 键确认选中项。
  * **视觉反馈**：聚焦按钮显示白色外框 + 绿色发光效果，文字清晰可见。
* **封面蛇装饰优化**：
  * 开始画面浮动蛇头新增侧面单眼（深色眼珠 + 白色高光），与游戏内蛇眼风格统一。

### 修复 (Fixed)
* **名称输入框光标问题**：修复点击输入框时光标不显示的问题（click 事件冒泡导致焦点被抢夺）。

### 优化 (Optimized)
* **界面标签精简**：移除游戏结束画面的”难度标签”（因游戏中可随时切换难度，单一标签无法定义整局）。
* **食物生成算法优化**：
  * **边缘规避**：食物尽量避开画布边缘（距离墙壁至少 1 格）。
  * **强制内圈机制**：一旦食物生成在边缘，后续 4 个食物强制只在内圈生成。
  * **边缘动态隔离**：任意两个边缘食物之间至少间隔 4 个内圈食物。
  * **安全兜底逻辑**：内圈全被蛇占据时自动回退到全局生成（防止死锁）。


## [1.1] - 2026-06-07

### 新增 (Added)
- **难度选择系统**：提供 `简单` / `普通` / `困难` 三档模式：
  - **简单**：移动速度慢（280ms），每食物计 1 分。
  - **普通**：移动速度中（220ms），每食物计 2 分。
  - **困难**：移动速度快（140ms），每食物计 3 分。
- **动态交互**：
  - 暂停时可点击按钮切换难度，速度和得分倍率即时生效。
  - 支持键盘快捷键 `1` / `2` / `3` 快速切换难度。
- **界面与UI**：
  - 画布左上角实时显示当前难度和得分倍率。
  - 游戏结束画面增加本局所选难度的展示。
  - 右侧历史记录列表的每条记录，均标注难度标签（通过绿/黄/红颜色区分）。

### 修复 (Fixed)
- 修复因事件冒泡重复触发，导致连按暂停键时无法正常暂停的 Bug。

## [1.0] - 2026-06-07

### 核心功能 (Core Features)
- **控制系统**：支持 `方向键` 和 `WASD` 键控制蛇的移动。
- **核心逻辑**：
  - 吃到食物时蛇身变长，分数对应增加。
  - 蛇头撞到墙壁或自身时判定游戏结束。
  - 支持 `空格键` 暂停或继续游戏。
- **流程控制**：提供独立的 `开始` 和 `重新开始` 按钮。

### 界面与视觉效果 (UI & Visuals)
- **主题风格**：采用深色科技感主题（翡翠绿蛇身 + 琥珀色食物）。
- **动效细节**：
  - 蛇身支持渐变色，并为蛇头绘制发光眼睛。
  - 食物具备脉冲呼吸动效。
- **粒子特效**：新增“吃食物”和“死亡”粒子炸裂特效。
- **数据与面板**：
  - 右侧设立独立的得分历史记录面板。
  - 接入 `localStorage` 实现最高分本地持久化存储。
- **状态反馈**：画布边框根据游戏状态变色（进行中为绿，死亡为红）。
