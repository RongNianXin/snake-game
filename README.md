# 🐍 贪吃蛇 (Snake Game)

> HTML5 Canvas 经典贪吃蛇游戏，支持多难度、玩家系统、排行榜与粒子特效。

<br>

## 🎮 游戏玩法

控制蛇吃掉食物，蛇身变长，得分增长。撞墙或撞到自己则游戏结束。

### 操作控制

| 按键 | 功能 |
|------|------|
| `↑` `↓` `←` `→` / `W` `A` `S` `D` | 移动方向 |
| `Space` | 暂停 / 继续 |
| `P` | 暂停 / 继续 |
| `1` `2` `3` | 快速切换难度 |
| `Enter` | 菜单中确认选中项 |
| `↑` `↓` | 暂停菜单中切换选项 |
| `←` `→` | 结束菜单中切换选项 |

- **移动端**：底部方向按钮操控，或在画布上滑动控制方向。

### 难度系统

| 难度 | 速度 | 单食物得分 |
|------|------|------|
| 🟢 简单 | 280ms | 1 分 |
| 🟡 普通 | 220ms | 2 分 |
| 🔴 困难 | 140ms | 3 分 |

- 随着得分增加，游戏速度会逐渐加快；
- 暂停时可以随时切换难度。

### 玩家系统

- 开始界面输入玩家名称，支持 **随机命名**（骰子按钮）；
- 已参与的玩家**自动缓存**，下次可直接点击选择；
- 每局成绩按玩家记录到排行榜。

<br>

## ✨ 功能特性

- **三种难度**：简单 / 普通 / 困难，游戏中可随时切换
- **玩家名称系统**：自定义名称 + 本地缓存 + 快速选择
- **排行榜**：按玩家汇总最佳成绩，支持多玩家独立排行
- **键盘导航**：暂停和结束画面中可用方向键 + 回车操作菜单
- **粒子特效**：吃食物（金色粒子）和死亡（红色粒子）均有炸裂效果
- **视觉细节**：
  - 蛇身渐变着色 + 蛇头双眼高光
  - 食物脉冲呼吸动效
  - 画布边框随游戏状态变色（进行中绿，死亡红）
  - 封面浮动蛇头装饰 + 侧面单眼
- **数据持久化**：最高分、历史记录、玩家列表均通过 `localStorage` 保存
- **响应式布局**：桌面双栏 + 移动端自适应
- **触屏适配**：移动端方向按钮 + 画布滑动手势

<br>

## 🛠 技术栈

| 技术 | 用途 |
|------|------|
| **HTML5 Canvas** | 游戏场景渲染（蛇、食物、粒子、网格） |
| **Vanilla JavaScript (ES6+)** | 游戏逻辑、数据管理、事件处理，零框架依赖 |
| **Tailwind CSS (CDN)** | UI 组件快速构建 |
| **Font Awesome 6** | 图标系统 |
| **Google Fonts** | Space Grotesk 字体 |
| **localStorage** | 客户端数据持久化（分数、记录、玩家列表） |
| **CSS3 Animations** | 背景动画、按钮动效、UI 过渡 |

## 📁 项目结构

```
snake-game/
├── index.html        # 唯一页面：HTML + CSS + JS 全部内联，开箱即用
├── CHANGELOG.md      # 版本变更日志
└── README.md         # 本文件
```

> 整个游戏为**单文件架构**，通过 `<style>` 和 `<script>` 标签内嵌在 `index.html` 中，无需构建工具，浏览器直接打开即可运行。

<br>

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/RongNianXin/snake-game.git

# 直接用浏览器打开
cd snake-game
open index.html       # macOS
start index.html      # Windows
```

或者下载 `index.html`，双击用浏览器打开即可开始游戏。
