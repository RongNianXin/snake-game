# 🐍 贪吃蛇 (Snake Game) v2.0

> HTML5 Canvas 诗词汉字版贪吃蛇 —— 吃汉字、集诗句、通关水墨结算。支持多难度、玩家系统、排行榜与自动化跑测。

<br>

## 🎮 游戏玩法

控制蛇吃掉食物，蛇身变长，得分增长。撞墙或撞到自己则游戏结束。

### 操作控制

| 按键 | 功能 |
|------|------|
| `↑` `↓` `←` `→` / `W` `A` `S` `D` | 移动方向 |
| `Space` | 暂停 / 继续 |
| `P` | 暂停 / 继续 |
| `1` `2` `3` | 快速切换难度 |
| `Enter` | 菜单中确认选中项 |
| `↑` `↓` | 暂停菜单中切换选项 |
| `←` `→` | 结束菜单中切换选项 |

- **移动端**：底部方向按钮操控，或在画布上滑动控制方向。

### 难度系统

| 难度 | 速度 | 单食物得分 |
|------|------|------|
| 🟢 简单 | 280ms | 1 分 |
| 🟡 普通 | 220ms | 2 分 |
| 🔴 困难 | 140ms | 3 分 |

- 随着得分增加，游戏速度会逐渐加快；
- 暂停时可以随时切换难度。

### 玩家系统

- 开始界面输入玩家名称，支持 **随机命名**（骰子按钮）；
- 已参与的玩家**自动缓存**，下次可直接点击选择；
- 每局成绩按玩家记录到排行榜。

<br>

## ✨ 功能特性

- **三种难度**：简单 / 普通 / 困难，游戏中可随时切换
- **玩家名称系统**：自定义名称 + 本地缓存 + 快速选择
- **排行榜**：按玩家汇总最佳成绩，支持多玩家独立排行
- **键盘导航**：暂停和结束画面中可用方向键 + 回车操作菜单
- **粒子特效**：吃食物（金色粒子）和死亡（红色粒子）均有炸裂效果
- **视觉细节**：
  - 蛇身渐变着色 + 蛇头双眼高光
  - 食物脉冲呼吸动效
  - 画布边框随游戏状态变色（进行中绿，死亡红）
  - 封面浮动蛇头装饰 + 侧面单眼
- **数据持久化**：最高分、历史记录、玩家列表均通过 `localStorage` 保存
- **响应式布局**：桌面双栏 + 移动端自适应
- **触屏适配**：移动端方向按钮 + 画布滑动手势

### 🎋 v2.0 诗词汉字版

- **汉字食物**：Canvas 动态渲染金色楷体汉字替代传统食物，带呼吸光晕
- **诗词收集**：从词库（静夜思 / 悯农 / 春晓）随机抽诗，5 个汉字依次吃入右侧田字格
- **去重生成**：食物智能去重，同一首诗不会重复出现相同汉字
- **归位结算**：集齐 5 字后田字格自动按诗句顺序重排，蛇身定格
- **水墨弹窗**：宣纸色通关弹窗 + 红色「詩」字印章 + 大字诗句展示，古风浓郁
- **无缝换关**：通关后长蛇不断、分数累加，无缝切换下一首新诗
- **自动化跑测**：内置 AI 寻路 + 倍速驱动 + 自动通关，支持 `?test=true` 或 `F8` 一键激活

## 🛠️ 自动化跑测与调试说明 (Development & Testing)

为了方便后续更复杂功能的稳定迭代，本项目内置了**自动化寻路跑测外挂（Auto-Play Test Mode）**。无需修改任何源码即可动态激活：

### 🚀 激活方式（两种途径）

1. **网址小尾巴参数法（最常用）**
   在浏览器中访问游戏时，只需在网址（URL）末尾加上特定的测试参数：
   * **正常游玩**：`http://.../word-snake.html`
   * **开启跑测**：`http://.../word-snake.html?test=true` （自动激活 AI 寻路及 8 倍速超速驱动）
2. **键盘后门快捷键法**
   在游戏运行的任意过程中，按下键盘上的 **`F8`** 键，即可对测试状态进行【开启 / 关闭】的实时取反切换，屏幕会有 Toast 提示。

### 🤖 自动化跑测闭环逻辑
1. **AI 寻路接管**：屏蔽键盘事件，小蛇根据食物汉字的坐标自动进行极简增量寻路。
2. **自动无缝下一关**：集齐 5 字触发结算弹窗后，测试模式延迟 `500ms` 自动模拟触发点击“下一关”按钮，实现无人值守的无限多局稳定性跑测。
